## 🍟 概述

* 基于.NETStandard 2.1 支持任何的.NET 版本。
* 常用方法封装：ParseToString,ParseToLongInt,ParseToLong,ParseToDouble...。
* 让.NET 开发更简单，更快捷，更容易。无任何第三方包引用。
